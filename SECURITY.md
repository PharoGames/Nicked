# Security Policy

## Supported Versions

Only the latest release of Nicked receives security fixes.

| Version | Supported |
|---|---|
| 1.0.x (latest) | Yes |
| Older versions | No |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Use [GitHub's private vulnerability reporting](https://github.com/PharoGames/Nicked/security/advisories/new) to disclose the issue confidentially.

Include as much detail as possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any proof-of-concept code or configuration
- Your suggested fix (optional)

You can expect an acknowledgement within **72 hours** and a resolution or status update within **7 days**.
